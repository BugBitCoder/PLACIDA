# 🌿 Placida

**Placida** is a human-centered mental health companion built using **design thinking principles**. It helps users track moods, manage stress, and access emotional support privately within the browser.

---

## ✨ About
Mental health support should feel approachable, safe, and easy to use. Placida aims to create that experience by combining emotional check-ins with lightweight wellness tools in one place, with **zero data collection**. Everything stays in your browser.

## 🎯 Features

### 😊 Mood Tracking
- emoji-based selection for fast logging.
- Add personal notes for context.
- Stored locally in `localStorage`.

### 📊 Personal Dashboard
- **Mood Trend Chart**: Visual 7-day history using Chart.js.
- **Streak Counter**: Encourages daily mindfulness.
- **Time-aware Greetings**: Personalized UI based on your time of day.

### 🌬️ Wellness Exercises
- **Guided Breathing**: 4-7-8 and Box breathing patterns.
- **Visual Cues**: Animated expansion/contraction for breath control.
- **Session History**: Track your progress over time.

### 💬 AI Chatbot Support
- **Supportive Conversations**: Scripted emotional support engine.
- **Quick-Reply Chips**: Faster interaction for common feelings.
- **Persistence**: Chat history is saved locally for follow-up.

### 📓 Weekly Summary
- **7-Day Review**: See your emoji trends at a glance.
- **Adaptive Prompts**: Journaling prompts that change based on your mood.
- **Private Journal**: Save your thoughts with automatic timestamps.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript.
- **Library**: [Chart.js v4.4.0](https://www.chartjs.org/).
- **Persistence**: `localStorage` (No backend database).
- **Icons**: Unicode Emojis.
- **Fonts**: [Google Fonts — Inter](https://fonts.google.com/specimen/Inter).

## 📁 Project Structure
- `index.html` — Landing page & Mood Logger.
- `dashboard.html` — Personalized stats and trends.
- `breathe.html` — Guided breathing exercises.
- `summary.html` — Weekly reflection and journals.
- `chatbot.html` — AI emotional support interface.
- `script.js` — Core application logic and storage.
- `features.js` — Extended feature engines (Chat/Breathe).
- `style.css` — Shared design system.

## 🚀 How to Run
1.  **Clone** the repository.
2.  Open `index.html` in any modern web browser.
3.  *(Optional)* Use the **Live Server** extension in VS Code for a better experience.
