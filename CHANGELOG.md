# 📋 PLACIDA — Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Week 1] — 6 Apr to 12 Apr 2026

### Added
- **Emoji-based Mood Logger**: Interactive mood selection on the primary landing page.
- **LocalStorage Data Persistence**: Secure local storage for mood logs and application state.
- **Personalized Dashboard**: Initial statistics display and mood history tracking.
- **Mood History List**: A chronological view of past mood logs for user reflection.
- **4-7-8 Breathing Exercise**: Core mindfulness tool with animated visual guidance.
- **Animated Respiratory UI**: Visual aids to assist users during guided breathing.
- **Weekly Summary Insights**: Automated calculation of emotional trends over a 7-day period.
- **Adaptive Journaling Prompts**: Context-aware prompts that change based on user mood.
- **AI Chatbot Prototype**: Basic conversational interface for emotional support.
- **Local Chatbot Engine**: Scripted response logic integrated into the frontend.

## [Week 2] — 13 Apr to 19 Apr 2026

### Added by Sahil
- **Mood Trend Chart**: Dynamic visualization of emotional data using Chart.js.
- **Streak Counter**: Motivation system with a "🔥" badge for daily engagement.
- **Time-aware Greetings**: Contextual greetings (Morning/Afternoon/Evening) based on local time.
- **Safe Data Reset**: A "Clear All Data" feature with a secure confirmation modal.

### Added by Ayushi
- **Quick-Reply Chips**: Faster interaction with 5 preset prompts in the chatbot.
- **Chat History Persistence**: Conversations are now saved locally using `localStorage`.
- **Chat Input Character Counter**: Real-time feedback on message length.
- **Mobile Hamburger Navigation**: Improved accessibility and UX for smaller screens.
